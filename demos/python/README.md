# Python Demo

This demo is the consumer side of the first sandbox proof.

- Install the STRling Python package from pip when possible.
- Load the canonical AST generated by the TypeScript demo.
- Compile it into a Python regex and validate the shared sample set.

## Commands

```bash
bash bootstrap.sh
.venv/bin/python consume_semver_ast.py
```

The consumer intentionally prints both the PCRE regex emitted by STRling and the Python `re` variant used for local execution. That keeps the emitter gap explicit instead of hiding it.
